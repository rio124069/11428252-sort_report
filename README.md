# 1. 氣泡排序 (Bubble Sort)
#  比較相鄰元素，將較大的逐一向後交換，如氣泡浮出。氣泡排序,O(n2),O(n2),O(1),穩定,簡單但效率低，適合小數據。
# 2. 插入排序 (Insertion Sort)
#  將資料逐一取出，插入已排序序列中的適當位置。插入排序,O(n2),O(n2),O(1),穩定,資料近乎已排序時效率極高。
# 3. 合併排序 (Merge Sort)
#  將資料對半拆解，遞迴排序後再合併成完整序列。合併排序,O(nlogn),O(nlogn),O(n),穩定,效能穩定，但較佔記憶體空間。
# 4. 快速排序 (Quick Sort)
#  選定基準點，將資料分為大小兩群，分治遞迴處理。快速排序,O(nlogn),O(n2),O(logn),不穩定,實務上平均速度最快，應用最廣。
# 5. 選擇排序 (Selection Sort)
#  從未排序區找出最小者，與未排序區的首位交換。選擇排序,O(n2),O(n2),O(1),不穩定,交換次數少，不受資料初始順序影響。
